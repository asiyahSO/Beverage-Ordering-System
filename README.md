# Beverage-Ordering-System (C++ Project)
> A console-based Beverage Ordering System developed using C++. This project simulates a real café ordering process where users can select drinks, customize options, and generate a final bill with membership discount and delivery charges.

# System Features
## Customer Input
- Customer name (first & last name)
- Phone number
- Number of orders

## Drink Ordering System
- Cup size selection (Small/Medium/Large)
- Drink Categories:
  - Coffee (Latte/Cappuccino/Americano)
  - Tea (Lemon Tea/Tea/Milk Tea)
  - Milk (Banana/Chocolate/Strawberry)
- Drink temperature selection (Hot/Normal/Cold)

## Pricing Features
- Automatic price calculation based on selection
- Membership discount (12%)
- Delivery charge (RM 5 COD / Pickup FREE)
